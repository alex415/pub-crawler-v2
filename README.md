# Pub Crawler
http://pub-crawler.herokuapp.com

## Overview
An Application to view photos of recommended bars by geographic area using the Google Places API, Google Maps API and Foursquare API.

## Installation
1. Clone repo
2. NPM install

## Technologies used:
* AngularJS
* Node.js
* Express
* Google Places API
* Google Maps API
* Foursquare API
